<Title>Matter</Title>
Matter is a design system built by <a href="https://khaos.systems">Khaos Systems</a> for the purpose of building
software and tooling for media production. - A relatively light wrapper around Radix-UX - Standardized styling meant for
productivity (hit boxes, sizes are sane for classical software-style apps) - Command Systems (/actions) - Shortcut
Systems - Lots of domain-specific components (e.g., color pickers) - Lots of build-in QoL features, like MtSelect, can
select options with 1-9 keys. Also hovering and pressing 1-9 will select without opening the select.

## Foundation Styles

Foundation styles include the core design tokens and base styles that form the building blocks of the design system. These are the atomic elements that ensure consistency across the UI and provide a solid base for building components.
